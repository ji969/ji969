## 项目简介

文绘 Agent 是一个基于多 Agent 协同的本地离线 AI 客户端，核心能力：

- **TXT 小说一键解析**：百万字长篇自动分片、角色抽取、关系网生成
- **双模式游玩**：原著分章节 Galgame 阅览 + 角色自由语音陪伴 Play
- **AI 素材全自动生成**：锁脸立绘、场景背景、角色专属语音
- **一键导出 Ren'Py 工程**：标准游戏工程输出，支持个人 / 商用双协议

## 核心技术栈

| 模块 | 技术选型 |
|------|---------|
| 前端引擎 | WebGAL + Electron |
| 多智能体调度 | LangGraph + FastAPI |
| 长文本离线推理 | Qwen2-7B-128K GGUF |
| 图像生成 | SDXL + IP-Adapter 锁脸 |
| 语音合成 | Fish Speech 离线本地部署 |
| 向量记忆 | Chroma 本地向量库 |
| 游戏工程导出 | Ren'Py 8 |

## 4 个核心 Agent

1. **长文本解析 Agent**：分片解析 → 实体识别 → 关系抽取 → 人设自检
2. **双模式游玩 Agent**：主线剧情阅览 + 自由 Play 对话，全本记忆防 OOC
3. **多媒体生成 Agent**：锁脸立绘（6 表情）+ 场景背景 + 专属语音合成
4. **创作者导出 Agent**：标准 Ren'Py 工程一键打包，双协议授权

## 差异化优势

| 维度 | 猫箱 / Talkie | 传统 Galgame | 普通 AI VN | 文绘 Agent |
|------|--------------|-------------|-----------|-----------|
| 百万字小说解析 | 无 | 人工编剧 | 仅短文本 | 分片增量解析 |
| 完整主线剧情 | 碎片化闲聊 | 单次制作 | 简单叙事 | 自动分章分 Part |
| 角色自由对话 | 有但无剧情 | 固定对话树 | 弱交互 | 长记忆防 OOC |
| 立绘 / 场景 / 语音 | 无 | 人工制作 | 简单配图 | AI 全自动生成 |
| Ren'Py 工程导出 | 无 | 需自己制作 | 部分支持 | 一键导出 |
| 使用门槛 | 低 | 极高 | 中 | 零门槛 |

## 目标用户

- **网文读者**（4 亿+）：把喜欢的小说变成可玩的视觉游戏
- **AI 角色陪伴用户**（千万级）：在全本剧情基底上与本命角色深度对话
- **同人创作者**：零美术预算快速产出可游玩的同人视觉小说
- **独立 Galgame 作者**：多 Agent 流水线接管 80% 素材工作
